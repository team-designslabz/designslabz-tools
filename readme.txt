== Description ==

**Designslabz Tools** plugin that customizes the WordPress admin by replacing the "Howdy" greeting, renaming "Post" to "Blog Post," adding custom admin footer text, and disabling the theme and plugin file editors.

**Features:**

* Replace “Howdy” with a cleaner “Logged in as” message
* Rename “Post” to “Blog Post” throughout the admin
* Add a custom footer message in the admin panel
* Disable theme and plugin file editors
* Uses PHP OOP principles with modular structure
* Follows WordPress coding standards

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/designslabz-tools` directory, or install it via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' menu.
3. All changes are applied automatically.

== Frequently Asked Questions ==

= Does this affect the frontend of my site? =
No. This plugin is strictly for backend admin interface enhancements.

= Can I revert the changes? =
Yes. Deactivating or deleting the plugin will return everything to default.

= Is this plugin multisite-compatible? =
It has not been fully tested on multisite environments.
